### Initial Findings:

#### Data & Screening:
- Over 460 criteria available for screening
- Uses FactSet data provider
- Point-in-time data to avoid survivorship bias
- RBICS classification system (replaced GICS)
- Custom universes for screening
- Historical data access for backtesting

#### Ranking Systems:
- Multi-factor ranking on 0-100 percentile basis
- Custom ranking system creation with weighted factors
- Pre-built models available
- Ranks predict returns quantitatively

#### Machine Learning (AI Factor):
- No-code AI/ML tool
- Algorithms: XGBoost, Neural Networks, Gradient Boosting
- Model tuning and evaluation tools
- Can predict stock performance
- Regression and classification models supported
- Grid search for hyperparameter tuning
- Categorical and macro features support (v1.1)

#### API Access:
- REST API available
- Python wrapper: p123api
- Programmatic access to data, screening, ranking, AI predictions
- Strategy management API (including rebalance)
- Factor List tool for downloading data for external ML

#### Ticker & Data Mapping:
- Handles delisted stocks
- Ticker changes tracked over time
- P123 ID system for securities
- Industry codes map to RBICS
- Mnemonic codes provided for convenience
