WORKFLOW SUMMARY (DROPBOX + GITHUB)
================================================================================

Data Storage:
  - Raw Excel files ‚Üí ~/Dropbox/Bond-RV-App-Data/Universe Historical/
  - Code ‚Üí GitHub (version controlled)
  - Parquet files ‚Üí Generated locally (not committed)

Setup on New Computer:
  1. Install Dropbox and ensure sync
  2. git clone https://github.com/ewswlw/Bond-RV-App.git
  3. pip install pandas pyarrow openpyxl
  4. python pipeline.py -i ~/Dropbox/Bond-RV-App-Data/Universe\ Historical/ -m override

Daily Workflow:
  - Add new Excel file to Dropbox folder
  - Wait for Dropbox to sync
  - Run: python pipeline.py -i ~/Dropbox/... -m append

After Code Updates:
  - git pull origin main
  - Run: python pipeline.py -i ~/Dropbox/... -m override
